# 🌸 Интернет-магазин цветов "Райский сад"

Интернет-магазин цветов с возможностью просмотра каталога, добавления товаров в корзину, оформления заказов и админ-панелью для управления товарами.

## 🚀 Функциональность

- ✅ Регистрация и авторизация пользователей
- ✅ Просмотр каталога товаров с карточками
- ✅ Добавление товаров в корзину
- ✅ Оформление заказов с указанием адреса
- ✅ Личный кабинет с историей заказов
- ✅ Редактирование профиля (аватар, имя, телефон, адрес)
- ✅ Админ-панель (добавление/редактирование/удаление товаров, управление заказами)
- ✅ Скидки на товары

## 🛠 Технологии

| Технология | Назначение |
|------------|------------|
| Python | Язык программирования |
| Flask | Веб-фреймворк |
| SQLite | База данных |
| SQLAlchemy | ORM для работы с БД |
| HTML/CSS | Интерфейс |
| Bootstrap | Стилизация |
| Jinja2 | Шаблонизатор |
